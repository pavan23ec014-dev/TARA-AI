# 🤖 TARA-AI

**TARA-AI** is an AI-powered desktop and engineering assistant built using **Python** and **local Large Language Models (LLMs)**. The goal of this project is to develop a personal AI assistant capable of understanding voice commands, performing desktop automation, assisting with engineering tasks, and supporting future AI integrations.

---

## 📌 Project Overview

TARA-AI is being developed as a modular AI assistant with support for:

- 🎤 Voice interaction
- 🧠 AI-powered conversations
- 📂 File searching
- ⚙️ Desktop automation
- 💻 Engineering assistance
- 🔒 Local LLM integration for privacy

This project is part of my learning journey in AI, Python, and software engineering.
